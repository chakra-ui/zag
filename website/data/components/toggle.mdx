- Can be controlled or uncontrolled

## Usage

Import the toggle package:

```tsx
import * as toggle from "@zag-js/toggle"
```

The toggle package exports two key functions:

- `machine` - State machine logic.
- `connect` - Maps machine state to JSX props and event handlers.

Then use the framework integration helpers:

<CodeSnippet id="toggle/usage.mdx" />

### Making it pressed by default

Use the `defaultPressed` property to make a toggle pressed by default.

```tsx {2}
const service = useMachine(toggle.machine, {
  defaultPressed: true,
})
```

### Controlled pressed state

Use `pressed` and `onPressedChange` for controlled usage.

```tsx
const service = useMachine(toggle.machine, {
  pressed,
  onPressedChange(pressed) {
    setPressed(pressed)
  },
})
```

### Listening for changes

When the toggle value changes, the `onPressedChange` callback is invoked.

```tsx {2-4}
const service = useMachine(toggle.machine, {
  onPressedChange(pressed) {
    console.log("toggle is:", pressed ? "On" : "Off")
  },
})
```

### Disabling the toggle

Set the `disabled` property in the machine's context to `true` to disable the
toggle.

```tsx {2}
const service = useMachine(toggle.machine, {
  disabled: true,
})
```

### Programmatic control

Use the API for imperative updates.

```tsx
api.setPressed(true)
```

## Styling guide

Each toggle part includes a `data-part` attribute you can target in CSS.

### Pressed State

When the toggle is pressed, the `data-state` attribute is applied to the root
and indicator with `on` or `off` values.

```css
[data-part="root"][data-state="on"] {
  /* styles for pressed state */
}

[data-part="root"][data-state="off"] {
  /* styles for unpressed state */
}
```

### Disabled State

When the toggle is disabled, the `data-disabled` attribute is added to the root
and indicator.

```css
[data-part="root"][data-disabled] {
  /* styles for disabled state */
}

[data-part="indicator"][data-disabled] {
  /* styles for disabled indicator */
}
```

## Accessibility

The toggle renders as a `button` with `aria-pressed` set to reflect the current
state.

### Keyboard Interactions

The toggle is a native button element, so standard button keyboard interactions
apply:

- <kbd>Enter</kbd> or <kbd>Space</kbd> toggles the pressed state.
